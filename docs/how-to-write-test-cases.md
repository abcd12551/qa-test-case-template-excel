# How to Write Test Cases in Excel

Writing structured test cases helps ensure better test coverage and consistent testing.

Typical test case structure:

Test Case ID  
Test Scenario  
Test Steps  
Expected Result  
Actual Result  
Status

Example:

TC_001  
Scenario: Verify login with valid credentials.

Steps:
1 Open login page  
2 Enter username  
3 Enter password  
4 Click login  

Expected Result:
User should be redirected to dashboard.
